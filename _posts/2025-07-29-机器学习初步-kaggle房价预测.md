## 1.数据处理

```python
train_df = pd.read_csv('path1')
```

### 1.3数据整合
```python
n_train = train_df.shape[0]
all_features = pd.concat((train_df.iloc[:, 1:-1], test_df.iloc[:, 1:]))
```
由于题目给出了train.csv和test.csv两个文件，处理的时候我们合并处理，以训练数据集的规模作为分割线，也就是`n_train`这个变量，shape[0]指的是它的行数，然后我们把这两坨文件接起来，这里就用的了pandas的`pd.concat`函数了，他的功能是把特定的部分拼接起来，记得使用`pd.iloc()`函数，这是因为训练集比测试集多了一个labels属性，但这确是需要统一的，所以我们把标签单独存到一个地方，把所有的features和对应的属性名存到一块

这里需要在回忆一下广播机制`[:]`表示全部的行/列，`[1:]`表示从第1行/列到最后一行，`[1:-1]`表示去除最后一行/列，广播机制中-1表示最后一个元素`[a:b]`里面的b处实际上是取不到的，也就是实际的取值范围是[a,b-1]
那么后面我们都用all_features作为所有数据的特征的存放的位置

### 1.4离散值和NaN的处理
在所给出的数据中，存在确实的数据和以文字(类别)形式的数据，下面做的就是把他们变成"可计算的数据"

```python
numeric_cols = all_features.select_dtypes(include=np.number).columns
all_features[numeric_cols] = all_features[numeric_cols].fillna(all_features[numeric_cols].median())
all_features = pd.get_dummies(all_features, dummy_na=True)
all_features = all_features.apply(lambda x: (x - x.mean()) / (x.std()))
all_features = all_features.fillna(0)
```
首先，我们使用`select_dtypes()`选出数据类型是数字的列并存到`numeric_cols`当中
然后把这些列当中的缺失值用中位数来补上(`fillna()`实现)，至于为什么要选中位数而不是平均数，请看原因：
![gm的回答](/assets/images/0729-1.png)
事实上，这与具体数据有关，因为这里存在一些非常极端的数据，所以平均数的意义被弱化了

接下来我们要对文字部分进行处理，这里用的是一种叫"独热编码"的方式

![独热编码](/assets/images/0729-2.png)
之所以不使用顺序编码，是因为顺序编码会导致没有的权重产生，比如红绿蓝三个类别，如果按照红-0、绿-1、蓝-2的方式编码，计算机会认为他们之间是有"大小关系的"

独热编码使用`pd.get_dummies()`函数，而`dummy_na=True`则表示对于缺失的值也当作单独的一类来编码
下一步我们对所有的数据进行特征缩放，是它们到同一尺度上

$$
x \leftarrow \frac{x - \mu}{\sigma},
$$

我们标准化数据有两个原因： 首先，它方便优化。 其次，因为我们不知道哪些特征是相关的， 所以我们不想让惩罚分配给一个特征的系数比分配给其他任何特征的系数更大(这一步对应了`apply()`函数)

最后的`fillna(0)`是为了防止一整列全是相同数字导致出现新的NaN

### 1.5张量化
我们最后使用这些数据训练的时候还是依托于pytorch架构，所以要转化为torch.tensor形式

```python
train_features = torch.tensor(all_features[:n_train].values, dtype=torch.float32)
test_features = torch.tensor(all_features[n_train:].values, dtype=torch.float32)
train_labels = torch.tensor(train_df.SalePrice.values.reshape(-1, 1), dtype=torch.float32)
```

一般而言标签(labels)是要以列向量的形式出现,故在选出特定列的内容之后使用reshape(-1,1)的函数

这个函数是很常用的，他最后得出的是一个类似于列向量的东西

## 2.模型搭建和训练

### 2.1初始化网络
采用了一个简单的MLP模型

```python
def get_net(in_features):
    net = nn.Sequential(
        nn.Linear(in_features, 256), 
        nn.ReLU(),                  
        nn.Linear(256, 1)           
    )
    def init_weights(m):
        if type(m) == nn.Linear:
            nn.init.xavier_uniform_(m.weight)
    net.apply(init_weights)
    return net
```
这里的神经网络只有一个隐藏层,然后这个`init_weights()`是对权重矩阵w的初始化方法
